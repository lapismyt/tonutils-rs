# tonutils

Pure Rust TON utilities and networking primitives inspired by `tonutils-go`.

This repository is under active reconstruction. The current milestone focuses on native ADNL TCP, LiteAPI, LiteBalancer, TL schema maintenance, TVM data structures, and smart-contract get-method support.
